## 2. 장르 분석

**비즈니스 실용서 + AI 기술서** 교차점에 위치한 책이다.

| 관점 | 컨벤션 |
|------|--------|
| 비즈니스 실용서 | 깔끔한 타이포, 1~2색 강조, 숫자/키워드 돋보이게 |
| AI/기술서 | 미니멀, 전문적, 네트워크·노드 시각 모티프 |
| 국내 IT 실용서 | 짙은 배경 + 밝은 제목 조합이 서점 섬네일 가시성 높음 |

**핵심 판단**: 제목 "101"이라는 숫자와 "팀장" 키워드가 임팩트의 원천. 표지에서 이 두 요소를 최우선으로 부각한다.

## 3. 메인안 A — "디지털 워커" 컨셉

### 컨셉 요약
"잠든 사이에 일하는 AI 직원"을 시각화. 에이전트 네트워크가 스스로 연결·작동하는 이미지.

### 시각 요소
- 배경: 짙은 네이비(#1B2A4A) 그라데이션 (상단 더 어둡게)
- 중앙 상단: 오픈클로 에이전트 아이콘 허브(원형) + 방사형 연결선 6개
- 연결 노드: 이메일, 캘린더, 슬랙, 리포트, 알림, 자동화 아이콘
- 연결선: 네온 블루(#4A9EDB) 점선 애니메이션 느낌
- 하단 1/3: 제목 텍스트 영역

### 타이포그래피
| 요소 | 설정 |
|------|------|
| 제목 1행 "오픈클로" | 흰색, 52pt, Bold |
| 제목 2행 "101가지 활용법" | 골드(#C08B2D), 44pt, Bold |
| 부제 | 밝은 회색(#B0BEC5), 18pt, Regular |
| 저자명 | 흰색, 14pt, 하단 배치 |

### 색상 팔레트
- 주색: `#1B2A4A` (네이비)
- 강조: `#C08B2D` (골드)
- 보조: `#4A9EDB` (스틸 블루)
- 텍스트: `#FFFFFF`, `#B0BEC5`

### 감정 톤
신뢰감 있고 전문적. 동시에 "이 도구는 실제로 일한다"는 역동성.

## 4. 대안 B — "101 임팩트" 컨셉

### 컨셉 요약
숫자 "101"을 표지 중앙에 압도적으로 배치. 풍부한 정보량과 즉시 활용 가능성을 강조.

### 시각 요소
- 배경: 밝은 흰색(#FFFFFF) + 상단 틸 그라데이션(#0D9488)
- 상단 1/3: 틸 배경에 "101" 초대형 타이포 (반투명 워터마크 느낌)
- 중앙: 제목 + 부제 텍스트 영역
- 하단: 아이콘 6개 가로 나열 (이메일·캘린더·슬랙·자동화·리포트·AI)
- 우측 하단: 저자명

### 타이포그래피
| 요소 | 설정 |
|------|------|
| "101" 배경 워터마크 | 틸 계열 연한색, 240pt, 반투명 |
| 제목 1행 "오픈클로" | 진한 네이비(#1F4E79), 48pt, Bold |
| 제목 2행 "101가지 활용법" | 틸(#0D9488), 38pt, Bold |
| 부제 | 회색(#6B7280), 16pt |
| 저자명 | 네이비, 14pt |

### 색상 팔레트
- 주색: `#FFFFFF` (화이트)
- 강조 1: `#0D9488` (틸)
- 강조 2: `#1F4E79` (네이비)
- 보조: `#6B7280` (회색)

### 감정 톤
에너지 넘치고 접근하기 쉬운. "101가지"라는 방대한 양에서 오는 자신감.
